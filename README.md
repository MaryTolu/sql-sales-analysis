# SQL Sales Analysis

## Overview
This is a self-initiated portfolio project created to practice SQL and business data analysis skills.

## Objective
The goal of this project is to analyze sales data and answer key business questions related to revenue, product performance, customer behavior, and sales trends.

## Tools Used
- SQL
- CSV dataset
- GitHub

## Business Questions
- Which products generate the most revenue?
- Which categories perform best?
- How do sales change over time?
- What is the average order value?
- Which customers or regions contribute most to sales?
